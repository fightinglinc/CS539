## CS539 Project
### Tech Stack
Frontend: HTML, CSS, JavaScript, jQuery, Bootstrap<br>
Backend: Flask, Restful API <br>
Database: MySQL <br>

### Install packakges
```
pipenv install
```

### Run the server
under the directory called backend, use the following command:
```
python app.py
```
The server will run on http://127.0.0.1:5000/.
